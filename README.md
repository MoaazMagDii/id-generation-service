# ID Generator (Java OOP Project)

A Java desktop application that demonstrates Object-Oriented Programming (OOP) principles through a configurable ID generation system with a graphical user interface (GUI).

---

## 🎯 Overview

This project was built as part of a college assignment to apply and practice core OOP concepts in a real-world scenario.

The application allows users to generate structured IDs based on customizable rules, while showcasing clean code organization and object-oriented design.

---

## 🧠 OOP Concepts Applied

- **Encapsulation** → Separation of data and behavior within classes
- **Inheritance** → Reusable base components for ID elements
- **Polymorphism** → Flexible handling of different ID generation strategies
- **Abstraction** → Clear interfaces for extending the system

---

## 🖥️ Features

- GUI-based interaction (no command line required)
